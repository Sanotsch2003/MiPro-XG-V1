# MiPro-XG-V1

- Big Endian
- 32-Bit
- 16 General Purpose Registers (GPR)
  - 0-13: "real" GPR
  - 14: Stack Pointer (SP)
  - 15: Program Counter (PC)
- Flags Register (FR)
  - 0: Negative (N)
  - 1: Zero (Z)
  - 2: Carry (C)
  - 3: Overflow (O)
